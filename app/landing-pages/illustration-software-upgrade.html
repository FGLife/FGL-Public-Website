Illustration Software Upgrade

FGL to Update Illustration Software on October 1, 2016

Great news! Our illustration software will be updated on October 1, 2016 and will offer many new enhancements and upgrades.

What Are the Enhancements & Upgrades?

Modernized Design: The updated illustration system will have a more modern look and feel. We recommend that you review the attached training videos to assist you in using the software.

Faster Download Speeds: You can expect a quicker speed in running illustrations, especially in “solve” routines.

Expanded Browser Support: The new updates provide expanded support for the latest browser versions, such as Microsoft’s Edge Browser.

Mobile Device Optimization: The illustration system will be easier to navigate on any device you are using: PC, laptop, tablet or phone.

Future Software Updates: Quicker download time for illustration software updates.

 
 

What Do You Need To Do?

Current Illustration Software Users

You need to run a software update between October 1, 2016 and October 26, 2016. Once this update is complete, you will see the new enhancements. If an update is not run by October 26, you will have to reinstall the software, which could be time-consuming and inconvenient. Avoid delays and update between October 1 and October 26.

New Software Users

If you do not currently have our illustration software downloaded, you can access this by visiting SalesLink and clicking on “illustration software” or click here for the link to download. If you download prior to October 1, 2016 you will still need to complete the update beginning October 1, 2016 and prior to October 26, 2016 for these new enhancements.
